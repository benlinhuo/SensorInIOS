# SensorInIOS
Collect all sensor data in iOS, including:
1. Accelerometer
2. Gyroscope
3. Magnetometer
4. Barometer
5. GPS
6. Bluetooth
7. MotionActivity observer
8. Audio decibel
9. Wifi status and strength
10. AmbientLight
11. Shake status and times
12. Steps/Pace/Distance
13.Prsusure

This app can also work in background with CoreLocation module.
